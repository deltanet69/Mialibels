### USER MANAGEMENT (RBAC)

Untuk akses user adalah sebagai berikut:
1. Super Admin & Kepsek
   - Akses ke semua halaman & Fitur
   - Bikin kelas
   - Input Nilai
   - Input Data Guru, Staff, Siswa
   - Input/Management data tabungan
   - Manage data users
   - Input/manage data Spp Sekolah
   - Delete Data
   - Edit Data
2. Guru & Staff
   - Akses ke halaman
     - Dashboard
     - Data Kelas
     - Input Data Guru, Staff, Siswa
     - Manage konten
     - Delete Data
     - Edit Data
3. Orang Tua
   - Akses ke halaman
     - Dashboard
     - Data Kelas
     - Data Guru
     - Data Staff
     - Data tabungan siswa
     - History Tabungan Siswa
     - Data Spp Sekolah
     - History Spp Sekolah
     - Tagihan Spp sekolah
